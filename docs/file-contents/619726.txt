urban gardening. Bestehende Urban-Gardening-Initiative innerhalb einer KGA.
Beispiel: Gemeinschaftsgarten.
Park / kleingartenpark. Öffentliche Funktion einer bestehenden KGA. Alternativ Ausstattung einer KGA 
mit Bänken, Pergolen oder anderen baulichen Elementen, die öffentlich genutzt werden. 
synergien. Bestehende Synergien auf gesamtstädtischer Ebene oder auf Quartiersebene vorhanden. 
Beispiel: Schulgarten, Patientengarten.
Hinweise zum grünsystem. KGA ist teilweise zu entwickelnder Teil des innerörtlichen Grünsystems (FREP 2017, 
LP 2030) und/oder liegt in der grüner Leitstruktur (vgl. Anhang 4. - Steckbriefe ‚Bedeutung im Grünsystem‘).
Hinweise zum lokalklima. KGA ist kaltluftproduzierende bioklimatische Entlastungsfläche und/oder Teil einer 
lokal oder übergeordnet wirksamen Kaltluftleitbahn (vgl. Anhang 4. - Steckbriefe ‚Lokalklimatische Relevanz‘).
Ökologische Hinweise. KGA mit wertvollen Strukturen wie alten Bäumen, Gegebenheiten im Umfeld wie 
einer Pufferfunktion zu Schutz- sowie anderen sensiblen Gebieten und/oder einer Biotopverbundfunktion 
(vgl. Anhang 4. - Steckbriefe ‚Ökologische Aspekte‘).
Hinweise zu schutzgebieten. KGA liegt in ein oder mehreren Schutzgebieten z.B. Landschaftsschutzgebiet (LSG) 
(vgl. Anhang 4. - Steckbriefe ‚Schutzgebiete‘) .
