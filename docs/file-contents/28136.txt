Annonyme Beisetzungen in Wolfartsweier 
 
Hiermit möchte ich die Anfrage stellen, ob es in naher Zukunft angedacht ist, in 
Wolfartsweier ein Feld auf dem Friedhof einzurichten, das annonyme 
Urnenbeisetzungen ermöglicht. 
 
Mein Vater wurde auf die Weise in Ettlingen bestattet, auf einer wunderschönen 
Wiese unter einem Baum. Das war sein Wunsch und ich finde es ist eine sehr 
schöne Idee.
