Stadtrat Cramer (KULT):  Ich möchte eine persönliche Erklärung für die Fraktion ab-
geben. Ist es jetzt in Zukunft von Ihnen so akzeptiert, dass man Anträgen, die Fraktio-
nen stellen, durch so einen Geschäftsordnungsantrag dann praktisch den Antragstel-
lenden keine Gelegenheit gibt, eine Wortmeldung zu machen. Wird das jetzt gute 
Übung des Hauses oder wie? Das ist so etwas Übles, das muss ich sagen. Es ist übel, 
was hier der Antragsteller der SPD rübergebracht hat. Aber gut. Wir werden es mit-
nehmen. 
 
 (Zurufe)
