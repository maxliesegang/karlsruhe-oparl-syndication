Kleinseeäckersiedlung: Schutz der Kinder auf dem Spielplatz 
 
 
 
1. Am Kickplatz wird ein Gitter installiert, um zu verhindern, dass Bälle auf den benach-
barten Kinderspielplatz geschossen werden. 
 
2. Auf dem Kickplatz wird eine zweite Torwand aufgestellt. 
 
 
 
 
Es  ist  unvermeidlich,  dass  Bälle  vom  Kickplatz  auf  den  in  der  Nähe  befindlichen  Kinder-
spielplatz   geschossen   werden,   wenn   keine   geeignete   Barriere   vorhanden   ist.   Dort 
spielende Kinder laufen Gefahr getroffen zu werden, bei kleinen Kindern kann dies zu Ver-
letzungen führen. 
 
Auf  dem  Kickplatz  gibt  es  nach  wie  vor  nur  eine  Torwand. Wenn  man  bedenkt,  dass  es 
sonst keinerlei Angebote für die Jugendlichen in der Siedlung gibt, sollte wenigstens eine 
zweite Torwand aufgestellt werden.
