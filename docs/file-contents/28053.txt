Preisgestaltung im Karlsruher Vierordtbad: Sondertarif für Babys und Kleinkinder 
 
Die Stadt Karlsruhe veranlasst, dass das Vierortbad an einem Tag unter der Woche 
für einige Stunden einen speziellen Kindertarif für 0 bis 3-Jährige anbietet. 
 
Das   städtische   Vierordtbad  hat   als   älteste   Badeanstalt   Karlsruhes   eine   lange 
historische  Tradition.  Im  Laufe  der  Jahre  hat  sich  das  Vierordtbad  zu  einem 
Wellnessbad  entwickelt,  dessen  Zielgruppe  erwachsene  Personen  sind,  die  sich 
einen   ruhigen   Badeaufenthalt   wünschen.   Daher   gibt   es   im   Vierordtbad   keine 
reduzierten  Eintrittspreise  für  Kinder.  Im  Rahmen  des  Karlsruher  Bäderkonzepts, 
welches    verschiedene    Bäder    auf    verschiedene    Bedürfnisse    der    Besucher 
zugeschnitten hat, ist ein zielgruppenorientiertes Konzept durchaus nachvollziehbar.  
 
Dennoch  erscheint  es  im  Hinblick  auf  ein  kinderfreundliches  Karlsruhe  und  in 
Anbetracht  vieler  in  der  Nähe  des  Bades  ansässiger  junger  Familien  geboten,  dass 
das  Vierordtbad  zumindest an  einem  Wochentag  für  einige  Stunden  über  die 
Mittagszeit einen Kindertarif für 0 bis 3-jährige anbietet.
