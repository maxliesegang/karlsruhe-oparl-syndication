Diakonische Suchthilfe Mittelbaden gGmbH
Sachverhalt | Begründung
Im Projekt Regenbogen der Diakonischen Suchthilfe Mittelbaden wird seit 25 Jahren mit Kindern und Jugend-
lichen, die aus schwer suchtbelasteten Familien stammen, gearbeitet. Das Angebot soll fortgeführt und aus-
geweitet werden bspw. auf die wichtige Elternarbeit. Damit einer größeren Anzahl von betroffenen Kindern 
geholfen werden kann, wird eine Zuschusserhöhung vom Träger beantragt, die die unterzeichnenden Frakti-
onen unterstützen.
