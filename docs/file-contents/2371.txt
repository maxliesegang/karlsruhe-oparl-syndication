Ergänzende Erläuterungen                                                                               Seite 2  
 
 
Begründung: 
Das Wildschweingehege ist ein touristisches Wanderziel, das eine hohe Bedeutung 
für das Naherholungsgebiet rund um unsere Ortsteile hat. Gerade dieses Gehege 
stellt eine besondere Einrichtung dar, weil es, bedingt durch die unmittelbare Be-
gegnung mit der Tierwelt, eine hohe Attraktivität besitzt. In unseren Ortsteilen feh-
len aber diesbezügliche Hinweisschilder auf das Wildschweingehege.  
 
 
 
 
 
 
 
 
 
Antrag an den Ortschaftsrat 
 
Der Ortschaftsrat beauftragt die Ortsverwaltung zuprüfen, welche Möglichkeiten be-
stehen mittels einer Beschilderung auf das Wildschweingehege im Horfwald hinzu-
weisen.
