Sachverhalt | Begründung
Während Besitzer*innen einer KVV-Jahreskarte ein Jobticket von der Stadt erhalten, ist dies bei Besitzer*innen 
der Jahreskarte der DB nicht der Fall. Die Stadt ist dringend darauf angewiesen, ihr Image als attraktive Arbeit-
geberin weiterzuentwickeln. Die bisherige Regelung stellt eine Ungleichbehandlung da. Ein Jobticket für DB-
Jahreskartenbesitzer*innen fördert das Umsteigen vom Auto auf eine klimafreundliche Beförderung.
