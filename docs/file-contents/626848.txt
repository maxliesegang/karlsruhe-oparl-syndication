Kurzfassung 
Die Verwaltung hat Bedenken hinsichtlich der Führung von Radverkehr auf der linken Seite. Es wird 
befürchtet, dass es über die Dornwaldsiedlung hinaus zu "Geisterfahrten" im Radverkehr über die 
Autobahnzu- und abfahrten kommt. 
Die Verwaltung wird prüfen, ob es möglich ist, auf diese Bedenken planerisch zu reagieren. Die Ergebnisse 
werden dem Ortschaftsrat mitgeteilt. 
 
Der Antrag wird als erledigt betrachtet.
