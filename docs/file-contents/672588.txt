 Sachverhalt | Begründung 
Die Mittel sollen für die Beschaffung der erforderlichen Technik zum Einsatz eines Scan-Cars eingesetzt wer-
den (Sensor-Set, Digitale Kartierung, Digitalisierung von Ausnahmegenehmigungen in Bewohnerparkzonen).  
Die Anschaffungskosten können erfahrungsgemäß durch die erwarteten Einnahmen aus einer verbesserten 
Einhaltung der Parkticketpflicht sowie aus Bußgeldern vollständig ausgeglichen werden. Das Verkehrsministe-
rium geht von Einnahmen von mindestens 300.000€ jährlich aus.
