Bolzplatz im Regenrückhaltebecken "Ob den Gärten" 
 
Der Ortschaftsrat beantragt die Verwaltung, im Regenrückhaltebecken im 
Neubaugebiet „Ob den Gärten“ einen Bolzplatz anzulegen. 
 
 
Begründung 
 
Das neue Regerückhaltebecken bietet sich von seiner Größe ideal zur Nutzung eines 
Bolzplatzes an. Daher soll das Regenrückhaltebecken so angelegt werden, dass 
Teile hiervon als Bolzplatz genutzt werden können. 
 
Dieser Bolzplatz würde den in unmittelbarer Nähe befindendem Kinderspielplatz 
ergänzen und das Freizeitangebot für Kinder im Neubaugebiet erweitern.  
 
Ein vergleichbarer im Regenrückhaltebecken „Michelsberg“ angelegter Bolzplatz 
wurde über Jahre hin von Kindern aus der unmittelbaren Nachbarschaft stark 
frequentiert.
