  Sachverhalt | Begründung 
Von Seiten der Diakonie liegt eine Bedarfsanalyse sowie ein Konzept für eine Beratungsstelle im Bereich Pros-
titution in der Stadt Karlsruhe vor. Im Sommer 2014 wurde diese Beratungsstelle für Prostituierte eröffnet.  
Die Anlaufstelle sowie die aufsuchende Sozialarbeit wurden bisher durch zeitlich befristete Projektmittel ver-
schiedener Träger finanziert. Damit die Diakonie das Angebot dauerhaft sichern kann, benötigt sie die Bereit-
stellung von städtischen Mitteln.
