Erhaltungssatzung für die Wettersteinstraße 
 
Antrag: 
Der    Ortschaftsrat    möge    beschließen,    das    Stadtplanungsamt    zu 
beauftragen,    zur    Sicherung    des    ortstypischen    Straßenbildes    der 
Wettersteinstraße eine Erhaltungssatzung aufzustellen. 
 
Begründung:  
In letzter Zeit wurden an der Steinkreuzstraße mehrere Häuser gebaut, die 
erheblich  vom  bestehenden  Bauvolumen  abweichen.  Dadurch  wurde  das 
Ortsbild wesentlich verändert.  
Um vorzubeugen, dass in der Wettersteinstraße durch „Bausünden“  der 
Charakter  des  Straßenbildes  im  größeren  Stil  verändert  wird,  sollte  eine 
Erhaltungssatzung  augestellt  werden,  die  bei  einem  Umbau  bzw.  einer 
Nachverdichtung, die Möglichkeiten aufzeigt.
