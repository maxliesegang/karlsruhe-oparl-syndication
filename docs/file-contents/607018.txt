Ergänzende Erläuterungen                                                                               Seite 2  
 
 
Es gibt mehrere Sprechanlagen im Weiherhofbad. Diese wären: eine Gegensprechanla-
ge an den Drehkreuzen, eine Anlage um vordefinierte Alarmmeldungen im Gefahrenfall 
abzuspielen und eine Sprechanlage für individuelle Durchsagen. 
 
Nach einem Test wurde festgestellt, dass die Sprechanlage für die individuellen Durch-
sagen einen Defekt aufweist. Dies wurde jedoch auf Grund der Tatsache, dass diese 
Anlage so gut wie nie genutzt wird, nicht zeitnah festgestellt. Nach Rücksprache mit 
der Betriebsleitung des Weiherhofbades wurde von den Vereinen keine Anfrage bzw. 
kein Hinweis auf die defekte Anlage gegeben. Da der Defekt nun bekannt ist, wird eine 
Instandsetzung beauftragt.
