1. Wurde zwischenzeitlich ein Unternehmen gefunden, das eine Anwendung zur 
Anmeldung und Datenverarbeitung der Anmeldungen bei Kindertagesstätten 
und Kindergärten erstellt – wie dies ursprünglich durch smartKITA erfolgen sollte 
– und ab wann kann diese genutzt werden? 
 
Im Rahmen des Projekts zur Findung einer neuen Software-Lösung zum Betrieb eines so-
genannten KITA-Portals wurde mittlerweile das Pflichtenheft fertiggestellt. Nach verschie-
denen Kostenschätzungen muss nach vergaberechtlichen Erwägungen von einer europa-
weiten Ausschreibung ausgegangen werden.  
 
Von zwei Unternehmen lagen individuelle Angebote vor, die zwar interessant, nach recht-
licher Würdigung jedoch nicht freihändig vergabefähig waren.  
 
Die Ausschreibung soll deshalb zum nächst möglichen Zeitpunkt auf den Markt gebracht 
werden, um verwertbare Angebote zu erhalten. Bewertung, Auswahl, Implementierung, 
Schulung und Einbindung von 43 Kindertagesstättenträgern werden danach einige Zeit in 
Anspruch nehmen. Nach der derzeitigen Projektplanung kann eine neue Software voraus-
sichtlich zum Kindergartenjahr 2018/2019 eingesetzt werden. 
 
Inwieweit eine Software gefunden werden kann, die auch statistische Auswertungen hin-
sichtlich der Bedarfsplanung ermöglicht, muss zunächst offen bleiben. Im Rahmen des 
Projekts wird eine Prozessbetrachtung zeigen, wie diese Aufgabe künftig effektiv erledigt 
werden kann. 
 
2. Wie wird momentan ein reibungsloses Verfahren bei der Anmeldung für Kinder-
tagestätten und Kindergärten gewährleistet? 
 
Wie in den vergangenen drei Vergabezyklen kann auch künftig das bisherige Verfahren 
smartKITA eingesetzt werden. Unerwünschte Funktionalitäten beziehungsweise Fehler 
können wie bisher per Einzelvergabe durch die Software Entwicklerfirma ProfiAG behoben 
werden. So wurden erst im Oktober 2016 Programmanpassungen für den E-Mail-Verkehr 
von dort vorgenommen und per Einzelabrechnung abgewickelt.
