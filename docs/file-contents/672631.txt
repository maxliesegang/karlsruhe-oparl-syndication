 Diakonisches Werk 
 Sachverhalt | Begründung 
Quartiersarbeit wird insbesondere durch die Bürgervereine bereits geleistet. Nach einer Umstellungsphase in 
2026, für die die Stadt bereits eine Kürzung des Zuschusses um 20.000,- Euro an das Diakonische Werk vor-
sieht, kann der Zuschuss in 2027 zusätzlich um weitere 20.000 Euro verringert werden, so dass die Kürzung in 
2027 40.000,- Euro beträgt.
