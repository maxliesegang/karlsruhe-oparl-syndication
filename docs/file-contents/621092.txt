 Sachaufwendungen 
Beschaffung von Helmen für die Jugendfeuerwehr 
Es wird eine Sachkostenerhöhung um 12.000 Euro für die Beschaffung von Helmen für die Jugendfeuer-
wehr Karlsruhe beantragt. 
Die Verwaltung hat die Anschaffung bereits noch im Haushaltsjahr 2020 vorgesehen. Eine entsprechende 
Mittelumschichtung ist vom Fachamt noch vorzusehen. 
Der Antrag ist damit erledigt.
