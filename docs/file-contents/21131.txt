Kampagne "Dein Tag für Afrika"  
 
 
Das Bürgermeisteramt ist bereit, die hiesigen Schulen über die Kampagne zu informieren. 
Von einer Werbeaktion wird jedoch abgeraten.  
 
Seit Jahren werden die Schulen von einer Vielzahl von Wettbewerben, Projekten, Mitmach-
aktionen etc. überfrachtet, was zwangsläufig auch zu Unterrichtsausfällen führt. Erst vor 
Kurzem haben sich die Geschäftsführenden Schulleiter mit diesem Thema auseinanderge-
setzt und von weiteren Aktionen dieser Art abgeraten.  
 
Das Bürgermeisteramt ist gerne bereit, die Schulen über die Kampagne „Dein Tag für Afrika“ 
zu  informieren,  rät jedoch  von  einer Werbeaktion  ab.  Die  Schulen können  dann  selbst  ent-
scheiden, ob sie daran teilnehmen oder nicht.  
 
Das Bürgermeisteramt empfiehlt, den Antrag für erledigt zu erklären.
